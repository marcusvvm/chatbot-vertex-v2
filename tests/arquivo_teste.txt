## Perguntas Frequentes (FAQ)

### 1. O que é o RAG Engine?
O RAG Engine (Retrieval-Augmented Generation) é um serviço do Google Vertex AI que permite combinar a capacidade generativa de modelos de linguagem com a recuperação de informações de uma base de documentos. Isso permite que o assistente responda perguntas baseando-se em documentos específicos do usuário.

### 2. Quais formatos de arquivo são suportados?
A API suporta os seguintes formatos de documento:
- PDF (.pdf)
- Texto (.txt)
- Word (.docx)
- Markdown (.md)

O tamanho máximo por arquivo é de 25 MB.

### 3. Como funciona a autenticação?
A API utiliza autenticação JWT (JSON Web Token). Para acessar os endpoints protegidos, é necessário incluir um token válido no header Authorization da requisição.

Exemplo:
```
Authorization: Bearer seu_token_aqui
```

### 4. Quais são os presets disponíveis?
Os presets são configurações pré-definidas para o comportamento do chat:

| Preset    | Modelo           | Temperatura | Uso Recomendado       |
|-----------|------------------|-------------|----------------------|
| balanced  | gemini-2.5-pro   | 0.2         | Uso geral            |
| creative  | gemini-2.5-pro   | 0.5         | Explicações complexas|
| precise   | gemini-2.5-flash | 0.1         | Consultas rápidas    |
| fast      | gemini-2.5-flash | 0.2         | Baixa latência       |

## Notas Técnicas

- O sistema utiliza Google Vertex AI como backend de IA.

- A região do modelo Gemini é us-central1.
- O servidor de produção usa Gunicorn com workers Uvicorn.
